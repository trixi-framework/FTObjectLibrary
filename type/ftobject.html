procedure, public :: description => FTObjectDescription

  • public function FTObjectDescription(self)

    Returns a character string of length DESCRIPTION_CHARACTER_LENGTH that represents the object. the base class implementation returns an empty string. Note that if the description is too long, the expected string will be trunctated. In general, one wants to use printDescription.

procedure, public, NON_OVERRIDABLE :: copy => copyFTObject

  • private function copyFTObject(self) result(copy)

    Base class implemetation of the assignment function. Call this from within any subclasses copy assignment function. All FTObject's implementation does is set the reference count to one, implying no additional ownwership to the caller that is creating the copy.
